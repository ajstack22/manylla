Design Rationale

These icons maintain the minimalist aesthetic while conveying the concept of a ribbon latch or string-tie closure system commonly found on manila envelopes. The red color (#D32F2F) ties back to the branded "y" accent, creating a cohesive visual language. Each design can work as a standalone app icon, a UI element for secure/private sections, or as part of the logo system.


Icon #1 (Classic Y Ribbon) is recommended as it naturally forms the letter "Y" while clearly representing a tied ribbon, making it the perfect bridge between the typography and the envelope metaphor.
